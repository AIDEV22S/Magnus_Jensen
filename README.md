# Magnus_Jensen

## En app som är designad för att registrera och administrera klubbmedlemmar.

## Vad du kan göra i appen :
![image](https://user-images.githubusercontent.com/96913339/209453076-b62025f0-59a3-4cec-8f60-976e186a98ad.png)

## Se alla nuvarande medlemmar
## söka bland dem via medlemsnummer eller förnamn
## justera/ändra medlemsavgiftstatus
## visa adressen för en vald klubbmedlem (kan visa flera)
## Ta bort selekterad medlem
![image](https://user-images.githubusercontent.com/96913339/209453149-7e6ea073-1027-4f0b-a3f7-842090bf03a1.png)

## Registrera nya medlemmar genom att ange namn, startdatum, betald avgift, samt gatuadress uppgifter
## Går också att lägga till en ny adress till befintlig medlem
![image](https://user-images.githubusercontent.com/96913339/209662162-0f5c69e9-5e62-4d18-8297-bc07173febfb.png)

## Kör sportklubb.py för att starta programmet.
